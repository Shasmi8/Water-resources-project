Bhakra Nangal Project



This project was launched with the mutual co-operation.In 1948, this project was launched with an expenditure of Rs 248 crore and was completed in 1968.This is designed to harness the water from the Sutlej River for the benefit of the concerned states.Actually,the whole project was completed in June 1977.
