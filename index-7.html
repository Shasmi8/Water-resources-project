Chambal Project



The Chambal River Valley Project is a major river valley project of India.Under this project, three dams-Gandhi Sagar Dam,Rana Pratap Sagar and Jawahar Sagar Dam, five power stations and one big barrage have been built.This project was started in 1953.The total expenditure for this project was 230.82 crores.
